Coach Directory Policy

Last updated: February 17, 2026

This policy defines how directory listings are published, reviewed, and maintained.

Eligibility Requirements

  • All six curriculum modules passed.
  • Capstone passed under published rubric criteria.
  • Scope-safe profile language (coaching only, no therapy claims).
  • Valid location and service modality details.

Moderation and Verification Status

  • Active: Verified and reviewed within the last 180 days.
  • Probation: Verified but review age exceeds 180 days.
  • Expired: Verified status is older than 365 days; listing may be hidden until re-review.
  • Pending: Submitted and awaiting moderation/verification.
  • Rejected: Did not meet listing or scope requirements.
  • Archived: Removed from public display by reviewer/admin action.

Update and Correction Requests

Listing owners may request profile updates via info@theexecutivefunctioninginstitute.com. EFI may request supporting verification information before updates are published.

Consent and Data Use

Submitting a directory request grants EFI permission to publish professional profile data in the public directory. EFI does not publish private billing, full assessment, or private session records in directory listings.
