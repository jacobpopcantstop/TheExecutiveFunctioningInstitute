Module C: Intervention Frameworks

Applied guides to backward planning, time blindness adaptation, and metacognitive questioning loops.

Backward Planning Guide

Start from due date, map quality bar, and back-schedule deliverables into calendar-anchored milestones.

  • Define done-state
  • Set interim proof points
  • Install pre-commitments

Time Blindness Strategy Stack

Replace internal clock reliance with visible countdowns, start rituals, and transition alarms.

  • Dual timers (macro + micro)
  • Visual elapsed-time cues
  • Time-estimation rehearsal

Metacognitive Prompts

Use prompts that train planning, monitoring, and adaptation instead of directive dependence.

  • "What is your first visible step?"
  • "What could derail this?"
  • "How will you recover if it does?"
