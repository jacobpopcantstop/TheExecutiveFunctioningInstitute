Educator Implementation Toolkit

Practical EF tools for classrooms: assignment architecture, transition supports, and student self-management scaffolds.

Classroom Use

High-Value Educator Assets

Assignment Chunking Protocol

Break complex assignments into milestone checkpoints with visible completion criteria.

  • Define done-state quality bar first
  • Map interim proof points
  • Attach start cues for each step

Transition Scripts

Reduce transition loss with explicit verbal sequencing and visual timing prompts.

  • Pre-transition preview language
  • Countdown + first-action cue
  • Recovery script for off-track moments

EF Observation Rubric

Track initiation, persistence, and self-monitoring across contexts without relying only on grades.

  • Observable behavior markers
  • Context notes (noise/load/time)
  • Weekly trend snapshot format

Student Reflection Prompts

Build metacognition through short post-task reflections tied to strategy change.

  • What worked today?
  • Where did breakdown begin?
  • What changes tomorrow?
